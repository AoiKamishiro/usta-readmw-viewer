# Changelog

## [0.0.1-alpha1] - 2026-05-30

### Added

- `U-Stella ReadmeViewer` としての初回 alpha 版を追加。
- Unity Inspector 上で Readme アセットを表示する機能を追加。
- Readme 内のリンクを Inspector から開けるように対応。
- UnityPackage インポート後に Readme アセットを自動で選択する機能を追加。

### Changed

- パッケージ名を `jp.co.u-stella.readmeviewer` に変更。
- VPM 向けのパッケージ構成に更新。
- CI/CD 設定を `jp.co.u-stella.facetracking` に近い release/CI/VPM rebuild 構成に更新。
